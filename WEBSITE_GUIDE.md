# Wanderlust Travel Vlog Website - User Guide

## 🌍 Welcome to Wanderlust!

This is a modern, fully responsive travel vlogging website showcasing adventures from around the world. The website features a cinematic design inspired by top YouTube travel creators and professional tourism brands.

---

## 📄 Pages Overview

### 🏠 Home Page (`/`)
The homepage is your gateway to the travel journey, featuring:

- **Hero Section**: Large cinematic banner with inspiring headline and CTA buttons
- **Featured Destinations**: 4 beautiful destination cards (Bali, Santorini, Patagonia, Tokyo)
- **Latest Travel Vlogs**: 3 video cards with thumbnails, duration, and view counts
- **Travel Stories Blog**: 3 recent blog posts with images, excerpts, and read times
- **Upcoming Events Preview**: 2 highlighted upcoming events
- **Testimonials**: 3 audience reviews with 5-star ratings
- **Newsletter Signup**: Dark-themed subscription section

**Navigation Options:**
- "Watch Videos" → Links to video content
- "Explore Gallery" → Goes to Gallery page
- "View All Events" → Goes to Events page

---

### 🖼️ Gallery Page (`/gallery`)
A visual showcase of travel photography and videography:

**Features:**
- **Search Bar**: Search by location or title
- **Media Type Tabs**: Filter by All, Images, or Videos
- **Category Filters**: 7 categories (All, Beaches, Mountains, Cities, Adventure, Food, Culture)
- **Masonry Grid Layout**: Pinterest-style responsive grid
- **Lightbox**: Click any image to view it fullscreen
- **Hover Effects**: Reveal title, location, and play button (for videos)

**Content:**
- 15 stunning travel photos and video thumbnails
- Locations include: Maldives, Swiss Alps, Rome, Nepal, Bangkok, India, Bali, Chile, Tokyo, Santorini, Kenya, Iceland, Morocco, and more

### Interactions
- **Smooth Hover Effects**: Cards lift and scale on hover

- **CTA Buttons**: Multiple variants (Primary, Secondary, Outline, Text)

## 📱 Responsive Design

### Desktop (> 1024px)
- 4-column grid for destination cards
- 3-column masonry gallery
- Full navigation bar
- Large hero sections

### Tablet (768px - 1024px)
- 2-column layouts
- 2-column masonry gallery
- Adjusted spacing

### Mobile (< 768px)
- Single column layouts
- Hamburger menu
- Touch-optimized interactions
- Stacked forms and content
